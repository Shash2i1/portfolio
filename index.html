Face Emotion Detection using AI

This Python program is an example of real-time emotion detection from a webcam video feed using OpenCV and a pre-trained ML emotion detection model. It detects faces in the video frames, predicts the emotions of those faces, and overlays the detected emotions on the video stream.

Project Info

  • Project Name : Face Emotion Detection using AI

  • Language(s) Used : Python (OpenCV, Keras, NumPy)

  • Project Description : This project implements a real-time facial emotion detection system using a webcam. It utilizes OpenCV for face detection and a pre-trained deep learning model to classify emotions into categories such as Angry, Disgust, Fear, Happy, Neutral, Sad, and Surprise. The system captures frames from a video feed, detects faces, predicts the emotion for each face, and displays both the face and emotion on the screen.
